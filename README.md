# ShieldWall — Web Application Firewall Platform

A real deployable WAF proxy + security scanner. Scan any website for vulnerabilities, harden code, and route live traffic through a firewall.

## Features
- 🔍 **URL Scanner** — scans security headers, TLS, CORS, cookies, exposed files
- 🔧 **Code Hardener** — detects framework and generates security patches
- 🛡️ **WAF Proxy** — real reverse proxy that inspects and filters live traffic
- 🌐 **Protect Sites** — register any domain and route its traffic through ShieldWall
